# RUDN-Webdev-HW2
Реализация React-приложения в виде Kanban-доски

В рамках данного домашнего задания мной была реализована упрощённая версия Kanban-доски, предназначенной для отображения списка задач в зависимости от стадии их выполнения. Основной целью работы являлась отработка навыков проектирования интерфейса веб-приложения с использованием компонентного подхода в библиотеке React.

В процессе выполнения задания особое внимание было уделено декомпозиции пользовательского интерфейса на самостоятельные React-компоненты, что позволило повысить читаемость кода и упростить его дальнейшее расширение и поддержку. Также в ходе работы были применены ключевые возможности React и экосистемы, включая маршрутизацию одностраничного приложения (SPA), использование хуков для управления состоянием и жизненным циклом компонентов, а также элементы стейт-менеджмента.

Дополнительно была реализована интеграция со сторонними ресурсами для получения данных, а также использованы библиотеки компонентов для ускорения разработки и обеспечения единообразного визуального стиля интерфейса. В результате выполненной работы был получен функциональный прототип Kanban-доски, наглядно отражающий процесс выполнения задач и демонстрирующий практическое применение современных инструментов фронтенд-разработки.
